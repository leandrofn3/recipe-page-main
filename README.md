---

O principal foco deste projeto foi aplicar conceitos fundamentais de **HTML semântico** e **CSS moderno**, explorando:  

- Estrutura clara e bem organizada do conteúdo.  
- Uso de listas ordenadas e tabelas para informações acessíveis.  
- Estilização com cores, tipografia personalizada e espaçamento adequado.  
- Aplicação de boas práticas de organização de código e responsividade.
